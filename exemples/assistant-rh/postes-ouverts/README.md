# Postes ouverts

Ce dossier rassemble les offres d'emploi en cours de recrutement. L'assistant crée chacune d'elles au moment de la publication.

## Convention de nommage

```
YYYY-MM-DD_titre-du-poste.md
```

Exemple: `2026-03-15_developpeur-backend.md`

Lorsqu'une grille d'entretien accompagne le poste, elle se trouve dans:

```
YYYY-MM-DD_titre-du-poste_grille.md
```

## Cycle de vie

1. L'offre est créée ici au moment de la publication
2. Les candidatures associées sont conservées dans `candidatures/`
3. Une fois le poste pourvu, l'offre peut être déplacée ou archivée
