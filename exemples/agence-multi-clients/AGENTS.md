# Agence multi-clients (workspace)

Ce dossier est un workspace: il fédère plusieurs bases BASE, une par client, et n'en est pas une lui-même.

Pour travailler, ouvrez un client: `clients/dupont-conseil` ou `clients/martin-digital`.
Pour router entre les clients en ligne de commande: `base route "votre demande" --workspace base.workspace.json`.
